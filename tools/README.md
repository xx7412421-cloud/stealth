# Stealth Mail Tooling Ecosystem

This directory contains isolated mini-product workspaces for OSS tooling contributors.

## Release tiers

- 	ools/v1/ contains tools intended to launch with the mail product.
- 	ools/v2/ contains tools intended for later release.

## Audience groups

- individual/ contains personal productivity tools.
- 	eam/ contains organization and team collaboration tools.

Contributors must only work inside the folder assigned in their GitHub issue. Tooling work must not modify the main application shell, existing routing, mail rendering engine, inbox architecture, wallet core, Stellar integration core, database schema, or existing design system unless an issue explicitly says otherwise.
